This project is a compact RGB macro pad built around the Seeed Studio XIAO RP2040.
It uses diode-isolated push buttons for reliable key detection and SK6812 Mini addressable RGB LEDs for per-key lighting effects.

The design focuses on: (KEY FEATURES)

1.Programmable Multi-Button Input Panel

2.Multi-Layer Operation

3.KMK + CircuitPython Firmware

4.Reliable Input Handling.

5.Optional Rotary Encoder Support

6.Expandable & PCB-Friendly Design
